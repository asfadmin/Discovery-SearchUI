Viewing Search Results

looks_one
Scenes - the left column

A list of scene names and acquisition times that were returned from a search are displayed in the leftmost column of the results panel.

Note: In Vertex, a scene is considered to be a package containing all files, or products, that are related to a specific location and time.

  1. Click on the cart icon next to a scene name to add all the scene’s files to the download queue. The cart changes appearance when this is done.
  2. Click on the zoom icon next to a scene name to zoom-in to the scene’s location on the map.

To view more information about a scene, click on the scene in the left column and the Scene Detail (center column) and Files column (right) will populate.

looks_two
Scene Detail - the center column

The Scene Detail column (center) provides a more detailed description of the scene, including Start Date/Time, Beam Mode, Path, Frame, Flight Direction, Polarization, Absolute Orbit, and a browse image. Not all scenes will have all extra information.

  1. The 'Baseline Tool' button opens the ASF Baseline Tool, which is used for the creation of InSAR stacks.
  2. The 'Citation' button opens a new window with citation guidance for published works using data, imagery, or tools accessed through ASF.
  3. The 'More Like This' button creates a search based on the selected scene’s path and frame.
  4. Clicking the image thumbnail will open an image browser so you can view the image in more detail.
looks_3
Files - the right column

The Files column (right) displays a list of files available for the currently selected scene. You may download files immediately or add them to your download queue by clicking on the appropriate icon.
