Using The Download Queue

If you are logged into Vertex, Clicking on the cart icon in the header, labeled Downloads, will display the contents of your current download queue.

Within the download queue, the list of files you have selected to download is displayed with some basic information on each file, such as file type and size.

  1. File IDs (names) can be copied
  2. Files can be individually downloaded
  3. Items can be removed from the queue

At the bottom of the list of files is a footer bar that shows several buttons.

  1. Clear will clear all files from the queue. The option {{ 'RESTORE' | translate }} will be displayed if you wish to undo this action.
  2. {{ 'COPY_FILE_I_DS' | translate }} will copy the file names of all files in the queue for use elsewhere. For example, this list could then be pasted into the Scene List Search window.
  3. {{ 'DATA_DOWNLOAD' | translate }} is used to download multiple products, with either the Download Python Script (.py) option or Metadata (metalink) file option.
  4. Metadata Download is used to export the contents of the download queue to a CSV, KML, or GeoJSON file. The KML and GeoJSON files provided by this feature are compatible with the Geographic Search Import feature.
