Basic List Search

looks_one
Search types

Selecting 'List' from the Search Type dropdown menu, opens the Scene List Search window and allows you to enter a list of scene or file names.

  1. A list type of 'Scene' allows searching for specific scene names (granule names), and the results will include any files that are part of those scenes.
  2. The 'File' list type allows searching for specific file names (product names), and the results will only include exactly those files.
looks_two
Edit List

Clicking 'Edit List' opens the Scene List Search window so you can make changes to your list.

looks_3
Search Results

Once all parameters have been chosen, click SEARCH. Search results will appear in the footer area of the Vertex window, and on the map.

Note: The number of files that are predicted to match the current search parameters is displayed under the SEARCH button. If there are no predicted matches, the search button will be greyed out and display NO RESULTS.
