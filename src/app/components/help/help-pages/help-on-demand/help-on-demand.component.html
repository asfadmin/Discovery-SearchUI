On Demand Processing

View the On Demand Manual (HyP3) here.

looks_one
Perform A Search

Perform any Sentinel search which will return a list of files that could potentially be suitable for Radiometric Terrain Correction (RTC).

looks_two
Find A Suitable File

Scroll through the list of results and you see some files have an icon of three gray squares next to them. These are files that are eligible for additional On Demand Processing.

looks_3
Save File To On Demand Processing Queue

If you wish to perform RTC processing on the file, then click the icon to add that file to the On Demand Processing queue. You can add as many files as you wish but there is a limit of two hundred files per month that you can submit for processing.

looks_4
Set Processing Options And A Project Name

Based upon the data type you will have different job processing types available, each with their own processing options. Select the processingoptions you wish.

When you submit the job(s) you can enter a project name for easy reference when the processing is complete. You can also select to only submit certain processing types or all processing types. By selecting a subset of the processing types and entering a project name, you can associate different project names to the different processing types if that is useful for you.

looks_5
Submit And View Jobs

Submit all of the files you want for additional processing. After submitting your files, you can view the list of jobs via the On Demand search window and download them from there.
